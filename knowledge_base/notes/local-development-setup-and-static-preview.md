# Local Development Setup and Static Preview Commands
Local development uses `npm ci` for dependency setup and `npm run serve` to launch a static server at `http://127.0.0.1:4173`, matching smoke test expectations.

Related: [[development-and-deployment]], [[local-static-server-post-mock-for-smoke-tests]], [[linting-stack-eslint-stylelint-htmlhint]]

Primary commands: `npm ci`, `npm run serve`, `npm run check`.
