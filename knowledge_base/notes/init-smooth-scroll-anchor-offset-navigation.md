# initSmoothScroll Anchor Navigation with Fixed-Header Offset
`initSmoothScroll` intercepts in-page hash links and applies a navbar-height offset so section headings remain visible beneath the fixed header after navigation.

Related: [[runtime-architecture]], [[init-navbar-scroll-threshold-and-active-section-mapping]], [[mobile-menu-accessibility-and-escape-close]]

Implementation anchor: `initSmoothScroll(navbar)` in `script.js`.
